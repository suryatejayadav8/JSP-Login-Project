# JSP Login & Registration System

This project is a simple and responsive Login & Registration System developed using JSP, HTML, CSS, and Java.

## Features
- User Login
- User Registration
- Responsive Design
- Colorful User Interface
- Password Validation
- Session Management
- Error Handling
- JSP Based Web Application

## Technologies Used
- Java
- JSP
- HTML5
- CSS3
- Apache Tomcat
- Eclipse IDE

## How to Run
1. Import the project into Eclipse.
2. Configure Apache Tomcat Server.
3. Deploy the project.
4. Run the application in a web browser.
